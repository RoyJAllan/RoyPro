
App Name:	Elevate Horizon Connect – Event Management
Author:	Roy Allan
Project Purpose:	The Purpose of this project is to allow users to view, search and register for events with a accessible application.
Core Features:	View Events, Search Events,  select events and view further details, Register for events.
Tech Stack:	React Native paper, Visual Studio Code, Expo, Android emulator testing
Project Status:	V1

V1 Screens: 

•	Home
•	Events
•	Event Details
•	Register
•	Settings



Final Notes: Using the app in this version should be quite intuitive. Starting on the Home Screen, you have 3 options at the bottom of the screen.

Home (Ui reflects you are on the home screen or any screen you are currently on).

Events: Here, the current available events will be displayed. Above the events, you have a search bar.
Selecting an event will add a tick left of the event selected. From here, you are now able to tap Event details, which will take you to  the event details screen (Only if you have selected an event)

Event Details: After navigating from the Events screen, you now have a view of further details of the event you have selected. You can now tap the register button which will navigate you to the register screen.

Register: Here, the user can add their name to register for the selected Event. (Further Selected event details will be added to the screen in the future.

Settings Screen: Here you have a toggle option for sound (Work in progress) and for dark mode (Working across all screens)
